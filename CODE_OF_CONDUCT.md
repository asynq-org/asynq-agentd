# Code of Conduct

This project aims to be welcoming, practical, and respectful.

## Expected behavior

- Be constructive and specific.
- Assume good intent, but be honest about problems.
- Focus critiques on ideas, code, and behavior, not people.
- Help keep the project approachable for people who are still learning.

## Unacceptable behavior

- Harassment, intimidation, or personal attacks
- Discriminatory or demeaning language
- Deliberate disruption of discussions or reviews
- Publishing private information without permission

## Maintainer role

Project maintainers may moderate discussions, close unproductive threads, or remove content that makes the project less safe or less useful for others.

## Reporting

If something happens that should not be handled in a public thread, report it privately to the maintainer through a non-public channel.
